An Introduction to Statistics

Download .zip Download .tar.gz View on GitHub

Description.

This course provides a refresher on the foundations of statistical analysis. Practicals are conducted using the R commander package; which provides an accessible interface to the R statistical language.

    Aims.

    To enable bench scientists to perform their own statistical testing, and to be to interpret the results presented in peer-reviewed journals.

    Learning Objectives.

  • To be able to define the mean, median and standard deviation of a population
  • Know the assumptions required for a one-sample and two-sample t-test
  • Know when to apply a paired or independent two-sample t-test

    • Module Outcome.

  • To formulate null hypotheses and interpret the results
  • To perform simple statistical calculations and create graphs in R Commander

    • Software Requirements.

  • R > 3.0 and the Rcmdr package installed. See our installation guide for details
  • If you have trouble installing the software, please contact the course organiser for assistance
  • If you still cannot install the software after following the installation instructions, please download the VirtualBox software for your operating system, and we will supply you with a virtual environment that you can use to run the course
